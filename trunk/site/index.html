BeanIO is an open source Java framework for reading and writing Java beans, or plain old java objects (POJO's), from a stream. Using an XML mapping file, BeanIO can be easily configured to parse objects from an input stream and write formatted objects to an output stream. BeanIO is well suited for reading and writing flat files, and currently supports CSV, delimited and fixed length stream formats.

Features:
  • XML based field mapping for streams
  • Support for CSV, delimited and fixed length stream formats
  • Strict record ordering enforcement
  • Common field validation rules
  • Configurable error messages
  • Extensible type handling

The BeanIO project is hosted on Google Code
